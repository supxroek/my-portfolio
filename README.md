## Highlights

- Dark, minimal portfolio layout with strong typography and red accent details.
- Animated sections for About, Skills, Experience, Projects, and Contact.
- Bilingual UI support through `i18next`.
- Responsive layout optimized for desktop and mobile.
- Project cards and timeline content tailored for developer portfolio presentation.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Motion
- i18next / react-i18next
- Lucide React
- Radix UI primitives

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Structure

- `src/app` - main app layout and sections
- `src/app/components` - reusable portfolio sections and UI components
- `src/imports` - screenshot assets used in the README and app
- `src/styles` - global styles and theme files
